## Running a single test module:
```
$ cd MyProject
$ python -m unittest tests/test_basic.py
```

## Running a single test case or test method:
```
$ cd MyProject
$ python -m unittest tests.test_basic.SimplisticTest
```

## Running all tests:
```
$ cd MyProject
$ python -m unittest discover
```
